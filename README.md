 E-Commerce-Sales-Analytics-Dashboard-DirectQuery---Power-BI-

 Project Overview

This project demonstrates an end-to-end Power BI dashboard built using DirectQuery mode to analyze e-commerce sales and customer behavior in near real-time.

⚙️ Tools & Technologies
Power BI (DirectQuery)
MySQL
DAX (Data Analysis Expressions)
🧠 Key Features
Designed a star schema data model with fact and dimension tables
Built interactive dashboards with KPIs such as Total Revenue, Orders, and Customer Metrics
Created time intelligence metrics such as Month-over-Month Growth
Enabled cross-page filtering using synced slicers
Analyzed query performance using Performance Analyzer
⚡ DirectQuery Implementation
Data is not imported into Power BI
All visuals generate SQL queries dynamically from MySQL
Ensures real-time data analysis
📈 Dashboard Pages
1. Sales Overview
Revenue by Category
Sales Trend Over Time
Revenue by City
KPI Summary
2. Customer Analysis
Total, New, and Repeat Customers
Top Customers by Revenue
Customer Distribution by Location
🚀 Key Learnings
Difference between Import Mode and DirectQuery
Importance of star schema for performance
DAX optimization for query efficiency
Time intelligence using a Date Table
Performance tuning in DirectQuery environments
